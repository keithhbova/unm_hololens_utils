# HoloFenk

Source code for the UNM HoloFenk firefighting project. 


## Installation:

```bash
pip install --upgrade pip
pip install scikit-build==0.16.6
pip install opencv-python==4.2.0.32
pip install pyqt6
pip install flask
pip install onnxruntime
```
